# Space Pirates

This Project aims to create a Pygame. Submitted in Fulfillment of The Academic Requirements in Programming 02

# Game
Space Pirates is a high-energy, 2D survival-arcade game where the player assumes the role of a Yellow Spaceship purging hostile ships from deep space. Targeted for PC platforms. The game follows a core loop of intercepting enemy ships, managing a health bar, prevent enemies from reaching the bottom screen, and surviving increasingly difficult waves of pirates to prevent a total mission failure.
